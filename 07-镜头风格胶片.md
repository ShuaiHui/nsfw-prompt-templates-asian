### 🎞️ 风格滤镜（胶片模拟 + 电影镜头）

> **核心原理**：不同的胶片和镜头组合会带来完全不同的色彩、对比度、颗粒感和氛围。
> **使用方法**：每个 prompt 选择 **1 种胶片风格 + 1 种镜头风格**（或只选一种），放在风格标签之前。
> **效果**：让 AI 生成的图片从"数字感"变成"胶片感"或"电影感"。

### 🎲 胶片/镜头生成规则

**每次生成 prompt 时，从以下组合中随机选择：**

```
[胶片风格 1个] + [镜头风格 0-1个] = 风格滤镜
```

**随机性规则：**

- **日系场景**：优先选择 `Classic Negative, Astia, NOSTALGIC Neg.` + `Cooke, Angenieux`
- **电影场景**：优先选择 `Kodak Vision3, ETERNA` + `ARRI, Panavision, Zeiss`
- **复古场景**：优先选择 `Kodak Portra, Gold, Classic Chrome` + `vintage Angenieux, Cooke`
- **黑白场景**：优先选择 `ACROS, Tri-X, T-MAX` + `Leica, Zeiss`
- **商业场景**：优先选择 `Ektar, Velvia` + `ARRI, Zeiss, Sigma Art`

**⚠️ 注意：**

- 胶片和镜头风格放在 **风格标签之后、画质标签之前**
- 不要同时用太多（1-2个最佳）
- 胶片风格会显著影响色彩，选择要和场景氛围匹配
- 镜头风格会影响锐度和焦外，人像优先暖调镜头（Cooke, Canon L），冷峻优先（Zeiss, Nikon）
