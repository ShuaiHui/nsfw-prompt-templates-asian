## 🎞️ 风格·胶片·氛围（情色向）

> **核心原则**：风格决定情绪基调，胶片决定色彩语言，两者配合创造统一氛围。
> **装配位置**：`[场景] → [景别/视角/设备/焦段/构图（见01）] → [风格标签] → [胶片风格] → [电影镜头] → [天气氛围]`
> **装配位置说明**：本文档排在 01（景别构图）之后，承接 01 的设备/景别选型，继续往下走。

## ⚙️ 附录 B：生成规则

### 装配流水线

```
[场景] → [景别/视角/设备/焦段/构图（见01）] → [风格标签] → [胶片风格] → [电影镜头] → [天气氛围] → [画质强化（见01）]
```

### 联动逻辑

| 规则 | 说明 |
|------|------|
| **胶片→场景** | 暖调胶片→私密/情欲；冷调→医疗/科技；黑白→艺术/残酷 |
| **镜头→氛围** | 暖柔镜头→浪漫场景；锐利镜头→商业/高端；复古镜头→怀旧 |
| **天气→光线** | 晴天/黄昏→暖光胶片；夜晚→Cinestill；阴天→Classic Chrome |
| **风格→胶片** | 偷拍写实→低饱和/黑白；AV封面→高饱和；唯美→Pro 400H |

### 组合示例

**卧室浪漫：**
`boudoir photography + Kodak Portra + Cooke S4 + warm sunset light`

**夜店偷拍：**
`amateur photo + Cinestill 800T + Leica Noctilux + neon glow`

**复古情色：**
`JAV screenshot + Kodak Gold + Helios 44 + golden hour`

**艺术裸体：**
`nude art photography + Kodak Acros + Leica Noctilux + foggy atmosphere`
